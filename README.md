### 관심 분야 및 성장  
- 데이터 분석, 머신러닝에 대한 관심으로 다양한 프로젝트 수행  
- 선형 회귀, 결정 트리, 랜덤 포레스트, 시계열 예측 등 다양한 모델 직접 구현 및 실습

- 실무에 적용 가능한 분석 역량과 데이터 기반 의사결정 경험 축적
